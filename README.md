# C Programming Repository

This repository contains **C programming_basics learning journey**, organized by chapter. Each chapter has exercises, lot of comments and programs.

---

## **Repository Structure**

The repo is organized as follows:

- **00 chapter/** – Introduction to C programming
- **01 chapter_Vars_Cons/** – Variables, Constants
- **02 chapter_instructions_operators/** – Instructions, Operators
- **03 chapter_Conditionals/** – If, Else, Switch
- **04 chapter_Loops/** – for, while, do-while loops
- **05 chapter_Functions_Recursion/** – Functions, Recursion
- **06 chapter_Pointers/** – Pointers and memory management
- **07 chapter_Arrays/** – 1D and 2D arrays
- **08 chapter_Strings/** – String handling in C
- **09 chapter_Structures/** – Structs and user-defined types
- **10 chapter_File_IO/** – File input/output handling

## what i've learnt:
- C concepts: variables, loops, conditionals, functions, pointers, arrays, strings, structs, and file I/O.
- Writing modular programs, debugging errors, and building problem-solving skills.
